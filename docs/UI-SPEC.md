## 1. 流程节点进度条（StageProgress）

### 1.1 布局

```
  简历筛选         测评           技术面          HR面          Offer
    ●──────────────●──────────────●──────────────○──────────────○
                                    进行中
```

- 节点名称在圆点**上方**。
- 圆点之间用水平线连接（`●────●`）。
- 当前停留节点下方显示状态文字。

### 1.2 颜色与状态文字

| 节点状态 | 圆点颜色 | 色值 | 下方文字 | 文字颜色 |
|----------|----------|------|----------|----------|
| `passed`（沿途） | 绿色 | `#52c41a` | 无 | - |
| `ongoing` | 蓝色 | `#1677ff` | **进行中** | 蓝色 |
| `terminated` | 红色 | `#ff4d4f` | **流程终止** | 红色 |
| 全部通过（最后节点） | 绿色 | `#52c41a` | **流程结束** | 绿色 |
| `pending` | 灰色 | `#d9d9d9` | 无 | - |

### 1.3 连线规则

- `passed` → `passed`：绿色
- `passed` → `ongoing`：绿色
- `passed` → `terminated`：绿色
- `ongoing` → `pending`：灰色
- `terminated` → `pending`：灰色
- `pending` → `pending`：灰色

即：连线颜色取决于**左侧**节点状态，左侧为 `passed` 时绿色，其余灰色。

### 1.4 典型场景

**进行中（技术面是当前节点）：**

```
  简历筛选         测评           技术面          HR面          Offer
    ●──────────────●──────────────●──────────────○──────────────○
                                    进行中
                                 [通过] [终止]
```

- 简历筛选：绿色（passed）
- 测评：绿色（passed）
- 技术面：蓝色（ongoing）+ [通过] [终止]
- HR面、Offer：灰色（pending）

**流程终止（技术面终止）：**

```
  简历筛选         测评           技术面          HR面          Offer
    ●──────────────●──────────────●──────────────○──────────────○
                                    流程终止
```

- 简历筛选：绿色（passed）
- 测评：绿色（passed）
- 技术面：红色（terminated）+ "流程终止"
- HR面、Offer：灰色（pending）

**全部通过：**

```
  简历筛选         测评           技术面          HR面          Offer
    ●──────────────●──────────────●──────────────●──────────────●
                                                                 流程结束
```

- 全部绿色（passed）
- 最后节点下方"流程结束"

### 1.5 交互规则

仅当前 `ongoing` 节点显示操作按钮：

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 通过 | 点击 [通过] 按钮 | 当前节点 → passed，下一节点 → ongoing |
| 终止 | 点击 [终止] 按钮 | 当前节点 → terminated，流程结束 |
| 撤回 | 点击已通过的绿色节点 | 弹确认 → 该节点 → ongoing，后续 → pending |
| 恢复 | 点击已终止的红色节点 | 弹确认 → 该节点 → ongoing |

## 2. 面试组卡片（主页 `/`）

```
┌──────────────────────────────────────────────┐
│  OfferTrack                     [设置] [⏻]   │
├──────────────────────────────────────────────┤
│                                              │
│  ┌────────────┐ ┌────────────┐ ┌──────────┐  │
│  │ 26年日常实习│ │ 27年暑期   │ │ 秋招提前批│  │
│  │            │ │            │ │          │  │
│  │  12 条记录  │ │  5 条记录   │ │ 8 条记录  │  │
│  └────────────┘ └────────────┘ └──────────┘  │
│                                              │
│  ┌────────────┐                              │
│  │ + 新建面试组│                              │
│  └────────────┘                              │
└──────────────────────────────────────────────┘
```

- 卡片网格，响应式列数。
- 每组卡片：组名 + 记录数 + 右上角下拉菜单（重命名/删除）。
- 默认组显示"默认"Tag，无删除菜单。
- 最后一张为"新建面试组"入口卡片（虚线边框）。

## 3. 记录卡片

```
┌──────────────────────────────────────┐
│ 字节跳动                   [删除]     │
│ 前端开发实习生                        │
│                                      │
│  简历筛选  测评  技术面  HR面  Offer    │
│   ●───────●───────●──────○──────○     │
│                   进行中              │
│                                      │
│ [26年日常实习]           2026-04-30   │
└──────────────────────────────────────┘
```

- 标题：公司名 + 删除按钮。
- 副标题：岗位名。
- 中部：StageProgress 进度条。
- 底部：所属组 Tag + 更新时间。
- 已结束记录半透明（opacity: 0.55），不可点击。

## 4. 记录详情页（`/record/:id`）

```
┌──────────────────────────────────────────────┐
│  ← 返回                                       │
│                                               │
│  公司名  [字节跳动]  ✏️                         │
│  岗位    [前端开发实习生]  ✏️                    │
│  所属组  [26年日常实习 ▾]                       │
│                                               │
│  ─────────── 面试流程 ───────────               │
│                                               │
│   简历筛选     测评      技术面     HR面   Offer  │
│     ●─────────●─────────●────────○──────○      │
│                          进行中                  │
│                       [通过] [终止]              │
│                                               │
│  ─────────── 节点排期 ───────────               │
│  技术面 · 2026-05-15 14:00 · 视频 · 60分钟 [编辑]│
│                                               │
│  创建时间：2026-04-15  更新时间：2026-05-01       │
└──────────────────────────────────────────────┘
```

- 基本信息：行内编辑（点击 ✏️ → 输入框 → 确认/取消）。
- 流程节点区：StageProgress + 操作按钮。
- 节点排期区：当前或已通过节点可设置排期（时间、形式、时长），用于日历展示。
- 不展示面经链接、自评分、轮次列表。
